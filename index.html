Trémaux's Algorithm

Trémaux's algorithm is a maze-solving method invented by Charles Pierre Trémaux. It is guaranteed to find a way out of any maze with well-defined passages, though it does not guarantee finding the shortest path. The algorithm relies on marking paths as they are traversed.

How it works:
  1. Mark each passage when first visited (hollow rectangle)
  2. If entering an unmarked passage, mark it and continue
  3. If all exits are marked twice, backtrack
  4. Never enter a passage marked twice unless no other choice
  5. Mark passages twice when backtracking (cross mark)
